# Password-generator

Title :Username and Password Generator with password strength checker.


Problem Statement: In Todays digital world, users create accounts on many websites and applications that nedd secure passwords to protect their information. Creating a strong and unique password everytime manually can be difficult.


Project Description:
 This Project generates username on the basis of username and the name of the application/websites they're creating account as well as secure random passwords and evaluates password strength.
 It uses cryptographically secure random generation and analyzes password security using modern techniques

 FEATURES:
 1. Random password generation
 2. Password strength evaluation
 3. Random Username generation

 Tech Stack:
    1.Python
    2.zxcvbn ( Password strength estimation)
